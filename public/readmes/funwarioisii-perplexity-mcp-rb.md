MCP Server for web search using Perplexity.

inspired by https://github.com/jsonallen/perplexity-mcp

## Usage

```json
{
  "mcpServers": {
    "perplexity": {
      "command": "/path/to/perplexity-mcp-rb/perplexity_mcp.rb",
      "env": {
        "PERPLEXITY_API_KEY": "pplx-xxxx"
      },
    }
  }
}
```
